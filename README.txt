Pretty Opt-In - Content Locker for Lead Generation

== Description ==

Pretty Opt In is a flexible WordPress Plugin make you can create the content locker (e.g. downloads, discounts, videos and so on) for lead generation.

https://www.youtube.com/watch?v=vuLSY7fJoCo

<h3>Presentation</h3>

<p><a href="https://1.envato.market/EQ1W4">Premium</a> | <a href="https://www.youtube.com/watch?v=vuLSY7fJoCo&list=PLYbs7RFTMsuIq1SaahhDJP0Wy1sd8iRlC/" >Videos Guides</a></p>

<p><strong>Premium Features:</strong></p>
<blockquote>
<ul>
<li>Customize lockers elements and design styles, include different layout, header title, locker message, button text and etc</li>
<li>Integrates with mail services and plugins, include MailChimp, SendGrid, SendinBlue, MailPoet, ActiveCampaign, FreshMail, MailJet, MailerLite</li>
<li>Choose locker subscription opt-in mode. Default mode is single opt-in</li>
<li>Show locker on mobile devices and registered members or not</li>
<li>Wrap the content you want to lock via pretty-locker shortcode</li>
<li>Use bulk actions to edit, publish, unpublish and delete lockers</li>
<li>Save the emails list of your subscribers on local wordpress database</li>
<li>Choose default sender name and sender email address for all of your outgoing emails.</li>
<li>Set locker cookie expired duration</li>
<li>Receive notifications via email when new lead received</li>
<li>Set custom link of Terms of Use & Privacy in your locker</li>
</ul>
</blockquote>

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `Pretty Opt In` from Plugins page.
